# API 参考

ZYX 提供 C++ 和 C API，用于嵌入到您的应用程序中。

## C++ API

C++ API 为 C++20 应用程序提供现代的、类型安全的接口。

- [数据库类](cpp-api) - 主数据库接口
- [事务类](transaction) - 事务管理
- [值类型](types) - 数据类型系统
- [错误处理](errors) - 异常处理

## C API

C API 提供与 C 兼容的接口，用于 FFI 绑定。

- [函数](c-api) - C API 函数
- [类型](c-types) - C 类型定义
- [错误码](c-errors) - 错误码参考
