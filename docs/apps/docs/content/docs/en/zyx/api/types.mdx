# Type System

Header: `include/zyx/value.hpp`

## zyx::Value

`zyx::Value` is a `std::variant` used for query parameters and result access:

| Alternative | C++ Type | Description |
|-------------|----------|-------------|
| Null | `std::monostate` | Absent / null value |
| Boolean | `bool` | `true` or `false` |
| Integer | `int64_t` | 64-bit signed integer |
| Float | `double` | Double-precision floating point |
| String | `std::string` | UTF-8 string |
| Node | `std::shared_ptr<Node>` | Graph node reference |
| Edge | `std::shared_ptr<Edge>` | Graph edge reference |
| Float vector | `std::vector<float>` | Compact numeric vector (used for embeddings) |
| String list | `std::vector<std::string>` | List of strings |
| List | `std::shared_ptr<ValueList>` | Heterogeneous list of values |
| Map | `std::shared_ptr<ValueMap>` | String-keyed map of values |

### Usage

```cpp
// Constructing values for query parameters
std::unordered_map<std::string, zyx::Value> params;
params["name"] = std::string("Alice");
params["age"] = int64_t(30);
params["active"] = true;
params["score"] = 95.5;

// Reading values from results
zyx::Value val = result.get("n.name");
if (std::holds_alternative<std::string>(val)) {
    std::string name = std::get<std::string>(val);
}
```

## zyx::Node

```cpp
struct Node {
    int64_t id;
    std::string label;                           // First label (backward compat)
    std::vector<std::string> labels;             // All labels
    std::unordered_map<std::string, Value> properties;
};
```

| Field | Description |
|-------|-------------|
| `id` | Internal node ID (assigned by `IDAllocator`) |
| `label` | Primary label (first in the list) |
| `labels` | All labels assigned to this node |
| `properties` | Key-value property map |

## zyx::Edge

```cpp
struct Edge {
    int64_t id;
    int64_t sourceId;
    int64_t targetId;
    std::string type;                            // Relationship type
    std::unordered_map<std::string, Value> properties;
};
```

| Field | Description |
|-------|-------------|
| `id` | Internal edge ID |
| `sourceId` | ID of the source node |
| `targetId` | ID of the target node |
| `type` | Relationship type (e.g., `"KNOWS"`) |
| `properties` | Key-value property map |

Note: Edge uses `type` (not `label`). Node uses `labels` (plural, can have multiple).

## graph::PropertyValue

Used internally by the query engine for property storage and `QueryBuilder`:

```cpp
struct PropertyValue {
    using MapType = std::unordered_map<std::string, PropertyValue>;

    PropertyValue();                    // Null
    PropertyValue(bool value);
    PropertyValue(int64_t value);
    PropertyValue(double value);
    PropertyValue(const std::string& value);
    PropertyValue(std::vector<PropertyValue> vec);  // List
    PropertyValue(MapType map);                      // Map

    PropertyType getType() const;
    std::string toString() const;
};
```

| `PropertyType` | Description |
|-----------------|-------------|
| `NULL_TYPE` | Null value |
| `BOOLEAN` | `bool` |
| `INTEGER` | `int64_t` |
| `DOUBLE` | `double` |
| `STRING` | `std::string` |
| `LIST` | `std::vector<PropertyValue>` |
| `MAP` | `unordered_map<string, PropertyValue>` |
| `DATE` | `TemporalDate` — calendar date (days since epoch) |
| `DATETIME` | `TemporalDateTime` — timestamp (milliseconds since epoch) |
| `DURATION` | `TemporalDuration` — ISO 8601 duration (months + days + nanos) |

## graph::ResultValue

Wraps a value returned by the query engine:

```cpp
class ResultValue {
    bool isNode() const;
    bool isEdge() const;
    bool isPrimitive() const;

    const Node& asNode() const;
    const Edge& asEdge() const;
    const PropertyValue& asPrimitive() const;

    std::string toString() const;
};
```

## Temporal Types

Header: `include/graph/core/TemporalTypes.hpp`

### TemporalDate

Calendar date stored as days since 1970-01-01.

```cpp
struct TemporalDate {
    int32_t epochDays = 0;

    static TemporalDate today();
    static TemporalDate fromISO(const std::string& s);   // "YYYY-MM-DD"
    static TemporalDate fromYMD(int year, int month, int day);

    int year() const;
    int month() const;
    int day() const;
    std::string toISO() const;  // "YYYY-MM-DD"
};
```

### TemporalDateTime

Timestamp stored as milliseconds since epoch.

```cpp
struct TemporalDateTime {
    int64_t epochMillis = 0;

    static TemporalDateTime now();
    static TemporalDateTime fromISO(const std::string& s);  // "YYYY-MM-DDTHH:MM:SS"
    static TemporalDateTime fromComponents(int y, int mo, int d, int h, int mi, int s, int ms = 0);

    int year() const;
    int month() const;
    int day() const;
    int hour() const;
    int minute() const;
    int second() const;
    std::string toISO() const;  // "YYYY-MM-DDTHH:MM:SSZ"
};
```

### TemporalDuration

ISO 8601 duration with three orthogonal components: months, days, and nanoseconds.

```cpp
struct TemporalDuration {
    int64_t months = 0;   // years * 12 + months
    int64_t days   = 0;   // weeks * 7 + days
    int64_t nanos  = 0;   // hours/minutes/seconds rolled into nanoseconds

    static TemporalDuration fromISO(const std::string& s);  // "P1Y2M3DT4H5M6S"
    static TemporalDuration fromComponents(int64_t years, int64_t months, int64_t weeks,
                                           int64_t days, int64_t hours, int64_t minutes,
                                           int64_t seconds, int64_t nanoseconds = 0);
    std::string toISO() const;
};
```

:::info
Temporal types support arithmetic in Cypher: `date + duration`, `datetime - duration`, `date - date` (returns duration), etc. Component access is available via property syntax: `d.year`, `dt.hour`, `dur.months`.
:::
