# API Reference

ZYX provides both C++ and C APIs for embedding in your applications.

The C++ API offers a modern, type-safe interface for C++20 applications.

- [Database Class](cpp-api) - Main database interface

- [Value Types](types) - Data type system

The C API provides a C-compatible interface for FFI bindings.
