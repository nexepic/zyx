# C API Reference

Header: `include/zyx/zyx_c_api.h`

All exported functions use the `ZYX_EXPORT` calling convention.

## Opaque Handles

| Type | Description | Freed by |
|------|-------------|----------|
| `ZYXDB_T*` | Database handle | `zyx_close()` |
| `ZYXResult_T*` | Query result handle | `zyx_result_close()` |
| `ZYXTxn_T*` | Transaction handle | `zyx_txn_close()` |
| `ZYXParams_T*` | Parameter set handle | `zyx_params_close()` |
| `ZYXList_T*` | List builder handle | `zyx_list_close()` |
| `ZYXMap_T*` | Map builder handle | `zyx_map_close()` |

## Database Lifecycle

### `zyx_open`

```c
ZYXDB_T* zyx_open(const char* path);
```

Open or create a database at `path`. Returns a database handle, or `NULL` on failure (call `zyx_get_last_error()` for details).

### `zyx_open_if_exists`

```c
ZYXDB_T* zyx_open_if_exists(const char* path);
```

Open a database only if the file already exists. Returns `NULL` if the file does not exist.

### `zyx_close`

```c
void zyx_close(const ZYXDB_T* db);
```

Close the database and release all resources.

### `zyx_get_last_error`

```c
const char* zyx_get_last_error(void);
```

Return the last error message for the calling thread. This is thread-local storage — safe to call from multiple threads.

## Query Execution

### `zyx_execute`

```c
ZYXResult_T* zyx_execute(const ZYXDB_T* db, const char* cypher);
```

Execute a Cypher query (auto-commit). Returns a result handle, or `NULL` on allocation failure.

### `zyx_execute_params`

```c
ZYXResult_T* zyx_execute_params(const ZYXDB_T* db, const char* cypher, const ZYXParams_T* params);
```

Execute a parameterized Cypher query.

### `zyx_result_close`

```c
void zyx_result_close(const ZYXResult_T* res);
```

Free a result handle. Must be called for every result returned by `zyx_execute*`.

## Transactions

### `zyx_begin_transaction`

```c
ZYXTxn_T* zyx_begin_transaction(ZYXDB_T* db);
```

Begin a read-write transaction. Returns `NULL` if another write transaction is active.

### `zyx_begin_read_only_transaction`

```c
ZYXTxn_T* zyx_begin_read_only_transaction(ZYXDB_T* db);
```

Begin a read-only transaction with snapshot isolation. Multiple read-only transactions can run concurrently.

### `zyx_txn_execute`

```c
ZYXResult_T* zyx_txn_execute(ZYXTxn_T* txn, const char* cypher);
```

Execute a Cypher query within a transaction.

### `zyx_txn_execute_params`

```c
ZYXResult_T* zyx_txn_execute_params(ZYXTxn_T* txn, const char* cypher, const ZYXParams_T* params);
```

Execute a parameterized Cypher query within a transaction.

### `zyx_txn_commit`

```c
bool zyx_txn_commit(ZYXTxn_T* txn);
```

Commit the transaction. Returns `true` on success. After commit (or rollback), the handle is still valid and must be closed with `zyx_txn_close()`.

### `zyx_txn_rollback`

```c
void zyx_txn_rollback(ZYXTxn_T* txn);
```

Roll back the transaction.

### `zyx_txn_is_read_only`

```c
bool zyx_txn_is_read_only(const ZYXTxn_T* txn);
```

Check if this is a read-only transaction.

### `zyx_txn_close`

```c
void zyx_txn_close(ZYXTxn_T* txn);
```

Free a transaction handle. Must be called for every transaction.

### List/Map Example

```c
// Build a list parameter: [1, 2, 3]
ZYXList_T* ids = zyx_list_create();
zyx_list_push_int(ids, 1);
zyx_list_push_int(ids, 2);
zyx_list_push_int(ids, 3);

ZYXParams_T* params = zyx_params_create();
zyx_params_set_list(params, "ids", ids);
zyx_list_close(ids);

ZYXResult_T* res = zyx_execute_params(db,
    "MATCH (n) WHERE id(n) IN $ids RETURN n", params);
zyx_result_close(res);
zyx_params_close(params);

// Build a map parameter: {year: 2024, month: 1, day: 15}
ZYXMap_T* dateMap = zyx_map_create();
zyx_map_set_int(dateMap, "year", 2024);
zyx_map_set_int(dateMap, "month", 1);
zyx_map_set_int(dateMap, "day", 15);

params = zyx_params_create();
zyx_params_set_map(params, "d", dateMap);
zyx_map_close(dateMap);

res = zyx_execute_params(db,
    "RETURN date($d) AS d", params);
zyx_result_close(res);
zyx_params_close(params);
```

## Complete Example

```c
#include <zyx/zyx_c_api.h>
#include <stdio.h>

int main() {
    // Open database
    ZYXDB_T* db = zyx_open("./testdb");
    if (!db) {
        printf("Failed to open: %s\n", zyx_get_last_error());
        return 1;
    }

    // Create nodes with transaction
    ZYXTxn_T* tx = zyx_begin_transaction(db);

    ZYXParams_T* params = zyx_params_create();
    zyx_params_set_string(params, "name", "Alice");
    zyx_params_set_int(params, "age", 30);

    ZYXResult_T* res = zyx_txn_execute_params(tx,
        "CREATE (n:Person {name: $name, age: $age})", params);

    if (!zyx_result_is_success(res)) {
        printf("Query failed: %s\n", zyx_result_get_error(res));
        zyx_txn_rollback(tx);
    } else {
        zyx_txn_commit(tx);
    }
    zyx_result_close(res);
    zyx_params_close(params);
    zyx_txn_close(tx);

    // Read query
    ZYXResult_T* read_res = zyx_execute(db,
        "MATCH (n:Person) RETURN n.name, n.age ORDER BY n.name");

    if (zyx_result_is_success(read_res)) {
        int cols = zyx_result_column_count(read_res);
        while (zyx_result_next(read_res)) {
            const char* name = zyx_result_get_string(read_res, 0);
            int64_t age = zyx_result_get_int(read_res, 1);
            printf("%s is %ld years old\n", name, (long)age);
        }
    }
    zyx_result_close(read_res);

    zyx_close(db);
    return 0;
}
```

## Resource Ownership

| Resource | Created by | Freed by | Notes |
|----------|-----------|----------|-------|
| `ZYXDB_T*` | `zyx_open` | `zyx_close` | Single owner |
| `ZYXResult_T*` | `zyx_execute*`, `zyx_txn_execute*` | `zyx_result_close` | String pointers from result are valid until `zyx_result_close` |
| `ZYXTxn_T*` | `zyx_begin_transaction`, `zyx_begin_read_only_transaction` | `zyx_txn_close` | Valid after commit/rollback until closed |
| `ZYXParams_T*` | `zyx_params_create` | `zyx_params_close` | String key/value pointers are copied internally |
| `ZYXList_T*` | `zyx_list_create` | `zyx_list_close` | Ownership transfers to params on `zyx_params_set_list` |
| `ZYXMap_T*` | `zyx_map_create` | `zyx_map_close` | Ownership transfers to params on `zyx_params_set_map` |
