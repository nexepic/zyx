# Advanced Queries

## 1) WITH as a Pipeline Boundary

```cypher
MATCH (p:Person)
WITH p, p.age AS age
WHERE age >= 30
RETURN p.name, age
ORDER BY age DESC;
```

## 2) UNION / UNION ALL

```cypher
MATCH (n:Person) RETURN n.name AS name
UNION ALL
MATCH (n:Company) RETURN n.name AS name;
```

## 3) UNWIND for Row Expansion

```cypher
UNWIND [1, 2, 3] AS x
RETURN x, x * 10;
```

## 4) CALL Subquery and Transactional Batching

```cypher
MATCH (n:Input)
CALL {
  WITH n
  CREATE (:Output {v: n.val})
} IN TRANSACTIONS OF 1000 ROWS
RETURN n.val;
```

## 5) LOAD CSV

With headers:

```cypher
LOAD CSV WITH HEADERS FROM 'file:///tmp/users.csv' AS row
RETURN row.name, row.age;
```

Custom delimiter:

```cypher
LOAD CSV WITH HEADERS FROM 'file:///tmp/users.tsv' AS row FIELDTERMINATOR '\t'
RETURN row.name;
```

## 7) EXPLAIN / PROFILE

```cypher
EXPLAIN MATCH (n:Person {name: 'Alice'}) RETURN n;
PROFILE MATCH (n:Person {name: 'Alice'}) RETURN n;
```
