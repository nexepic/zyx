# 属性索引

## 设计

`PropertyIndex` 不是单一树，而是“按值类型分树”：

- string / int / double / bool 各自 root map
- 每个 key 记录已推断类型

## 支持能力

- 精确匹配：`findExactMatch`
- 范围查询：`findRange`
- 复合索引：`createCompositeIndex`、`findCompositeExact`、`findCompositePrefix`

## 维护策略

- 写入：`addProperty` / `addPropertiesBatch`
- 删除：`removeProperty`
- 重建：`clearIndexData`
- 持久化：root map + key type map + composite state
