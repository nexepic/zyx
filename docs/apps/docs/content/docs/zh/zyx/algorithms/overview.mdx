# 算法总览

## 一、查询与优化

- 逻辑重写：谓词简化、过滤下推、投影下推
- 代价驱动：增强索引选择、Join 重排
- 位置：`src/query/optimizer/*`

## 二、索引与查找

- B+Tree 基础结构：`IndexTreeManager`
- 标签索引：`LabelIndex`
- 属性索引：`PropertyIndex`（含复合索引）
- 位置：`include/graph/core/IndexTreeManager.hpp`、`src/storage/indexes/*`

## 三、存储与空间管理

- 段链管理、空闲段复用、压缩/整理
- activity bitmap 跟踪槽位活跃性
- LRU 缓存降低热点读取开销
- 位置：`src/storage/*`

## 四、向量检索

- 距离函数（L2/IP）
- KMeans 训练码本
- Product Quantization 编码与近似距离
- DiskANN 图索引搜索
- 位置：`include/graph/vector/*`、`src/vector/*`

## 组合关系

这些算法不是孤立调用：查询优化会选择索引策略，索引读写又依赖存储结构；向量索引与普通图查询通过过程调用和算子集成在同一执行框架中。
