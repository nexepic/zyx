# 缓存淘汰算法

## 算法

当前通用缓存采用 LRU：

- 双结构：链表（访问顺序）+ 哈希表（定位）
- 命中后移动到链表头
- 容量满时淘汰链表尾部

## 并发特性

- `shared_mutex` 保护读写
- `tryPut` 支持非阻塞插入（锁竞争时直接跳过）

## 统计

缓存内置命中/未命中计数，可用于性能分析。
