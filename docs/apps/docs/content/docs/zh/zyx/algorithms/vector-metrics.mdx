# 向量度量

## 支持度量

`VectorMetric` 当前实现：

- L2 squared（欧式距离平方）
- Inner Product（以负值形式用于“越小越近”排序）

## 数据类型组合

- Float32 vs Float32
- Float32 vs BFloat16
- BFloat16 vs BFloat16

## 性能实现细节

- 关键循环采用手动展开，帮助编译器自动向量化
- 在 PQ、KMeans、DiskANN 检索中共享同一套距离计算路径

## 源码定位

- `include/graph/vector/core/VectorMetric.hpp`
- `include/graph/vector/core/BFloat16.hpp`
