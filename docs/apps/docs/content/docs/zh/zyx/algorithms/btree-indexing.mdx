# B+Tree 索引

## 核心对象

`IndexTreeManager` 是索引树通用引擎，支撑标签索引、属性索引和部分向量映射结构。

## 关键能力

- `insert` / `insertBatch`
- `remove`
- `find`
- `findRange`

## 结构维护

写入与删除会触发：

- 叶子分裂（splitLeaf）
- 父节点插入传播（insertIntoParent）
- 下溢再平衡（redistribute / mergeNodes）
- 根节点收缩

## 数据类型

键比较器按 `PropertyType` 选择（int/double/string/bool）。

- `include/graph/core/IndexTreeManager.hpp`
- `src/core/IndexTreeManager.cpp`
