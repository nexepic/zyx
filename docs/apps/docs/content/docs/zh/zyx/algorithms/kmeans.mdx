# K-Means 聚类

## 用途

当前 KMeans 主要服务于 PQ 训练，不是通用分析 API。

## 算法流程

- 随机初始化 centroid
- 迭代执行：
  - E-step：样本归簇
  - M-step：更新 centroid
- 直到收敛或达到最大迭代次数

## 并行实现

- 样本量足够大时，E-step 可并行
- 使用 thread-local 累加器降低锁竞争
- 最后合并线程局部统计量
