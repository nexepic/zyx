# 标签索引

## 作用

将标签字符串映射到实体 ID 集合，用于加速 `MATCH (n:Label)`。

## 组件

- `LabelIndex`：标签索引门面
- `IndexTreeManager`：底层 B+Tree
- `SystemStateManager`：保存 rootId 与 enabled 状态

## 行为要点

- 支持延迟启用 `createIndex()`
- `drop()` 会清数据并移除状态键
- 支持 batch 添加（`addNodesBatch`）

## 源码定位

- `include/graph/storage/indexes/LabelIndex.hpp`
- `src/storage/indexes/LabelIndex.cpp`
