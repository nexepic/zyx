# 乘积量化（PQ）

## 核心思想

把高维向量切分为多个子空间，每个子空间独立量化为 centroid 编号，降低存储和距离计算成本。

## 主要流程

1. `train`：每个子空间运行 KMeans 训练 codebook
2. `encode`：向量映射为每子空间一个 code
3. `computeDistanceTable`：预计算 query 到各 centroid 距离
4. `computeDistance`：用 code 在表中查距并累加

## 并行策略

- 子空间数量足够大时可并行
- 避免外层并行与内层 KMeans 争用导致死锁
