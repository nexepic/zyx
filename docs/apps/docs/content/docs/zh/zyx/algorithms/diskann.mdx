# DiskANN 算法

## 核心对象

`DiskANNIndex` 提供向量索引核心能力：

- `insert` / `batchInsert`
- `search(query, k)`
- `train(samples)`
- `remove`

## 配置

`DiskANNConfig` 关键参数：

- `dim`
- `beamWidth`
- `maxDegree`
- `alpha`
- `metric`
- `autoTrainThreshold`

## 搜索思路

- 采用图上的贪心/束搜索近似最近邻
- 可结合 PQ 距离表减少计算成本

## 与系统的集成

- 由 `VectorIndexManager` 管理生命周期
- 通过过程调用 `db.index.vector.queryNodes` 对外暴露

## 源码定位

- `include/graph/vector/index/DiskANNIndex.hpp`
- `src/vector/index/DiskANNIndex.cpp`
- `include/graph/storage/indexes/VectorIndexManager.hpp`
