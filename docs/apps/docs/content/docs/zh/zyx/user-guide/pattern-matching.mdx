# 模式匹配

## 单跳模式

```cypher
MATCH (a:Person)-[:KNOWS]->(b:Person)
RETURN a, b;
```

## 可选匹配

```cypher
MATCH (p:Person)
OPTIONAL MATCH (p)-[:WORKS_AT]->(c:Company)
RETURN p.name, c.name;
```

## 变长路径

```cypher
MATCH p = (a:Person)-[:KNOWS*1..3]->(b:Person)
RETURN p;
```

当前支持的变长形式包括：

- `*`
- `*N`
- `*N..M`
- `*N..`
- `*..M`

## 路径过滤建议

优先把过滤放在 `WHERE`，并尽可能使用可索引属性：

```cypher
MATCH (p:Person)-[:KNOWS*1..2]->(f:Person)
WHERE p.name = 'Alice' AND f.age >= 25
RETURN f;
```

## 多模式组合

```cypher
MATCH (a:Person)-[:KNOWS]->(b:Person), (b)-[:WORKS_AT]->(c:Company)
RETURN a.name, b.name, c.name;
```
