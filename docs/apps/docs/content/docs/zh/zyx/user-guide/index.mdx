# 用户指南

本指南面向“直接使用产品”的用户，重点是如何高效使用 ZYX，而不是源码实现细节。

| 1 | [快速开始](quick-start) | 构建、打开数据库、跑通首条查询 |

## 当前产品形态

- 以 CLI 为核心工作流：REPL、脚本模式、批量导入命令。
- Cypher 已覆盖常见读写子句、子查询、`LOAD CSV` 与管理 DDL。
- 支持图查询 + 向量检索 + GDS 过程。
- 事务满足 ACID，持久化基于 WAL。

## 文档约定

- CLI 可执行文件路径：`./buildDir/apps/cli/zyx`
- REPL 示例默认以 `;` 结束
- 事务语句为 `BEGIN` / `COMMIT` / `ROLLBACK`
- 特性边界以仓库根目录 `UNSUPPORTED_CYPHER_FEATURES.md` 为准
