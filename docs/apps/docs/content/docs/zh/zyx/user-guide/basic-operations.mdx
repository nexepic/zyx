# 基本操作

## 创建（Create）

```cypher
CREATE (p:Person {name: 'Alice', age: 30});
CREATE (c:Company {name: 'ZYX Inc'});
MATCH (p:Person {name: 'Alice'}), (c:Company {name: 'ZYX Inc'})
CREATE (p)-[:WORKS_AT {since: 2025}]->(c);
```

## 读取（Read）

```cypher
MATCH (p:Person) RETURN p;
MATCH (p:Person) WHERE p.age >= 30 RETURN p.name, p.age;
MATCH (p:Person)-[r:WORKS_AT]->(c:Company) RETURN p.name, c.name, r.since;
```

## 更新（Update）

```cypher
MATCH (p:Person {name: 'Alice'})
SET p.age = 31, p.city = 'Shanghai';

MATCH (p:Person {name: 'Alice'})
REMOVE p.city;
```

## 删除（Delete）

```cypher
MATCH (p:Person {name: 'Alice'})-[r:WORKS_AT]->()
DELETE r;

MATCH (p:Person {name: 'Alice'})
DETACH DELETE p;
```

## 基础索引与约束

```cypher
CREATE INDEX person_name_idx FOR (n:Person) ON (n.name);
SHOW INDEX;
DROP INDEX person_name_idx;

CREATE CONSTRAINT person_name_unique FOR (n:Person)
REQUIRE n.name IS UNIQUE;
SHOW CONSTRAINT;
DROP CONSTRAINT person_name_unique;
```

具体语法见 [高级查询](advanced-queries)。
