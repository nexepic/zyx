# 高级查询

## WITH 管道化

```cypher
MATCH (p:Person)
WITH p, p.age AS age
WHERE age >= 30
RETURN p.name, age
ORDER BY age DESC;
```

## UNION / UNION ALL

```cypher
MATCH (n:Person) RETURN n.name AS name
UNION ALL
MATCH (n:Company) RETURN n.name AS name;
```

## UNWIND

```cypher
UNWIND [1,2,3] AS x
RETURN x, x * 10;
```

## CALL 过程

当前内置过程包含（以实现为准）：

- `dbms.listConfig()` / `dbms.getConfig(key)` / `dbms.setConfig(key, value)`
- `dbms.showStats()` / `dbms.resetStats()`
- `algo.shortestPath(startId, endId)`
- `db.index.vector.queryNodes(indexName, topK, queryVector)`
- `db.index.vector.train(indexName)`

示例：

```cypher
CALL dbms.showStats() YIELD key, value
RETURN key, value;
```

## EXPLAIN / PROFILE

```cypher
EXPLAIN MATCH (n:Person {name: 'Alice'}) RETURN n;
PROFILE MATCH (n:Person {name: 'Alice'}) RETURN n;
```
