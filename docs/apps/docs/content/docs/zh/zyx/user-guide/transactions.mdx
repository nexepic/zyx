# 事务控制

## 事务语句

在 REPL 中直接输入：

```cypher
BEGIN;
CREATE (u:User {name: 'Alice'});
CREATE (u:User {name: 'Bob'});
COMMIT;
```

回滚：

```cypher
BEGIN;
CREATE (u:User {name: 'Temp'});
ROLLBACK;
```

## 行为要点

- 不支持嵌套事务。
- 活跃事务未提交时，离开作用域会自动回滚（API 场景）。
- 写事务是单写模型；并发读通过快照执行。

## 何时使用显式事务

- 多条语句必须原子提交
- 批量写入需要明确提交边界
- 需要在失败时统一回滚
