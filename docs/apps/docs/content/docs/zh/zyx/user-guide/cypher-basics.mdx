# Cypher 基础

## 基本模式

- 节点：`(n)`、`(n:Label)`、`(n {k: v})`
- 关系：`(a)-[:REL]->(b)`、`(a)-[r:REL]->(b)`
- 多标签：`(n:Person:Employee)`

## 常用子句

- 读：`MATCH`、`OPTIONAL MATCH`、`WHERE`、`RETURN`
- 写：`CREATE`、`MERGE`、`SET`、`REMOVE`、`DELETE`、`DETACH DELETE`
- 结果控制：`DISTINCT`、`ORDER BY`、`SKIP`、`LIMIT`
- 组合：`WITH`、`UNION`、`UNION ALL`
- 其他：`UNWIND`、`CALL`

## 参数化

```cypher
MATCH (n:User {name: $name}) RETURN n;
```

参数由 API 层注入（C++ `execute(query, params)` / C API `zyx_execute_params`）。

## 表达式能力（当前实现）

- 算术：`+ - * / % ^`
- 比较：`= != < <= > >= IN BETWEEN`
- 逻辑：`AND OR XOR NOT`
- 字符串匹配：`STARTS WITH`、`ENDS WITH`、`CONTAINS`
- 空值：`IS NULL`、`IS NOT NULL`
- 条件：`CASE WHEN ... THEN ... ELSE ... END`

## 常见函数类别

- 字符串：`upper/lower/substring/trim/...`
- 数值：`abs/ceil/floor/round/sqrt/sign`
- 聚合：`count/sum/avg/min/max/collect`
- 列表：`size/head/tail/last/range/coalesce`
- 量词：`all/any/none/single`
- 其它：`timestamp/randomUUID`

完整支持面建议同步查看仓库根目录 `UNSUPPORTED_CYPHER_FEATURES.md`。
