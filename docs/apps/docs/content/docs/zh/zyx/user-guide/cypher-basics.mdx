# Cypher 基础

ZYX 使用 Cypher 作为图查询语言。本页概述当前可用的子句、表达式与函数能力。

## 查询管线

一条典型 Cypher 查询按照以下管线执行：

```mermaid
flowchart LR
  A[MATCH / CREATE] --> B[WHERE]
  B --> C[WITH]
  C --> D[RETURN]
  D --> E[ORDER BY / SKIP / LIMIT]
```

:::info
并非所有子句都必须出现。最简查询可以只有一个 `RETURN`。上图的顺序反映了子句的典型组合方式。
:::

## 子句分组

| 分组 | 子句 |
|---|---|
| 读取 | `MATCH`, `OPTIONAL MATCH`, `WHERE`, `RETURN` |
| 写入 | `CREATE`, `MERGE`, `SET`, `REMOVE`, `DELETE`, `DETACH DELETE` |
| 结果整形 | `DISTINCT`, `ORDER BY`, `SKIP`, `LIMIT` |
| 组合与展开 | `WITH`, `UNION`, `UNION ALL`, `UNWIND`, `FOREACH` |
| 过程与子查询 | `CALL ...`, `CALL { ... }`, `CALL { ... } IN TRANSACTIONS` |
| 数据加载与诊断 | `LOAD CSV`, `EXPLAIN`, `PROFILE` |
| 管理 DDL | `CREATE/DROP INDEX`, `CREATE/DROP CONSTRAINT`, `SHOW INDEXES`, `SHOW CONSTRAINT` |

## 模式与表达式

| 构造 | 写法 | 说明 |
|---|---|---|
| 节点 | `(n)`, `(n:Label)`, `(n {k: v})` | 变量、标签、属性可任意组合 |
| 关系 | `-[r:REL]->`, `-[:REL]->`, `-[:REL]-` | 有向或无向 |
| 多标签 | `(n:Person:Employee)` | 同时匹配多个标签 |
| 变长 | `[:REL*1..3]` | 1 到 3 跳 |
| Map 投影 | `n {.name, .age, score: expr, .*}` | 选择性投影节点属性 |
| 列表推导 | `[x IN list WHERE cond \| expr]` | 过滤并转换列表 |
| 模式推导 | `[(n)-[:KNOWS]->(m) \| m.name]` | 将模式匹配结果投影为列表 |

## 运算符与谓词

| 类别 | 运算符 |
|---|---|
| 算术 | `+ - * / % ^` |
| 比较 | `= <> != < <= > >=` `IN` `BETWEEN ... AND ...` |
| 逻辑 | `AND OR XOR NOT` |
| 字符串 | `STARTS WITH` `ENDS WITH` `CONTAINS` `=~` |
| 空值 | `IS NULL` `IS NOT NULL` |
| 条件 | `CASE WHEN ... THEN ... ELSE ... END` |

| 数值 | `abs`, `ceil`, `floor`, `round`, `sqrt`, `sign`, `log`, `log10`, `exp`, `pow`, `sin`, `cos`, `tan`, `asin`, `acos`, `atan`, `atan2`, `rand`, `pi`, `e` |
| 列表 | `size`, `range`, `head`, `tail`, `last`, `reverse` |
| 类型转换 | `toInteger`, `toFloat`, `toBoolean` |
| 时间 | `date`, `datetime`, `duration` |
| 实体自省 | `id`, `labels`, `type`, `keys`, `properties` |
| 路径 | `nodes(path)`, `relationships(path)`, `length(path)`, `shortestPath(...)` |

## 时间类型

ZYX 支持三种时间类型：**Date**、**DateTime** 和 **Duration**。

### 创建时间值

```cypher
// 当前日期与时间
RETURN date() AS today, datetime() AS now

// 从 ISO 字符串创建
RETURN date("2024-01-15") AS d,
       datetime("2024-01-15T13:45:30") AS dt,
       duration("P1Y2M3DT4H5M6S") AS dur

// 从分量 map 创建
RETURN date({year: 2024, month: 1, day: 15}) AS d,
       datetime({year: 2024, month: 1, day: 15, hour: 13, minute: 45, second: 30}) AS dt,
       duration({years: 1, months: 2, days: 3, hours: 4}) AS dur
```

### 时间算术

| 表达式 | 结果类型 | 说明 |
|---|---|---|
| `date + duration` | Date | 日期加时长 |
| `datetime + duration` | DateTime | 时间戳加时长 |
| `date - duration` | Date | 日期减时长 |
| `datetime - duration` | DateTime | 时间戳减时长 |
| `date - date` | Duration | 两个日期之差 |
| `datetime - datetime` | Duration | 两个时间戳之差 |
| `duration + duration` | Duration | 两个时长之和 |
| `duration - duration` | Duration | 两个时长之差 |

### 分量访问

```cypher
WITH date("2024-06-15") AS d
RETURN d.year, d.month, d.day

WITH datetime("2024-06-15T13:45:30") AS dt
RETURN dt.year, dt.month, dt.day, dt.hour, dt.minute, dt.second

WITH duration("P1Y2M3DT4H5M6S") AS dur
RETURN dur.months, dur.days
```

## 统计聚合

除基本聚合函数外，ZYX 还支持统计聚合：

```cypher
// 样本标准差与总体标准差
MATCH (s:Student)
RETURN stDev(s.score) AS sampleStdDev,
       stDevP(s.score) AS populationStdDev

// 离散百分位数（返回集合中的实际值）
MATCH (p:Person)
RETURN percentileDisc(p.age, 0.5) AS medianAge

// 连续百分位数（线性插值）
MATCH (p:Person)
RETURN percentileCont(p.score, 0.75) AS p75Score
```

| 函数 | 说明 |
|---|---|
| `stDev(expr)` | 样本标准差（Welford 算法）。n < 2 时返回 NULL。 |
| `stDevP(expr)` | 总体标准差。n < 1 时返回 NULL。 |
| `percentileDisc(expr, p)` | 离散百分位数 — 返回排位 `floor(p * (n-1))` 处的值。 |
| `percentileCont(expr, p)` | 连续百分位数 — 相邻值间线性插值。 |

## 参数化查询

```cypher
MATCH (u:User {name: $name})
WHERE u.age >= $minAge
RETURN u.name, u.age;
```

:::tip 生产环境建议
使用参数 API 可以避免 Cypher 注入并提升计划缓存命中率：
- C++：`Database::execute(query, params)`
- C API：`zyx_execute_params(...)`
:::
