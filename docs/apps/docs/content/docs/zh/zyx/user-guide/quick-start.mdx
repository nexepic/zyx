# 快速开始

## 1) 构建项目

```bash
./scripts/run_tests.sh --quick
```

构建后 CLI 通常位于：

```bash
./buildDir/apps/cli/zyx
```

## 2) 创建数据库并进入交互模式

```bash
./buildDir/apps/cli/zyx database create ./demo.graph
```

打开已有数据库：

```bash
./buildDir/apps/cli/zyx database open ./demo.graph
```

## 3) 执行第一组语句

在 REPL 中输入：

```cypher
CREATE (u:User {name: 'Alice', age: 30});
CREATE (u:User {name: 'Bob', age: 25});
MATCH (a:User {name: 'Alice'}), (b:User {name: 'Bob'})
CREATE (a)-[:KNOWS {since: 2024}]->(b);
MATCH (a:User)-[r:KNOWS]->(b:User)
RETURN a.name, b.name, r.since;
```

## 4) REPL 常用命令

- `help`：查看帮助
- `save`：flush 到磁盘
- `debug ...`：查看内部存储状态
- `exit`：退出

## 5) 脚本模式执行

```bash
./buildDir/apps/cli/zyx database exec ./demo.graph ./seed.cypher
```

`seed.cypher` 中建议每条语句以 `;` 结束。
