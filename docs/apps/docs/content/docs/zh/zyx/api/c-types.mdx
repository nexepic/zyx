# C API 类型

头文件：`include/zyx/zyx_c_api.h`

## 不透明句柄

```c
typedef struct ZYXDB_T ZYXDB_T;          // 数据库句柄
typedef struct ZYXResult_T ZYXResult_T;  // 查询结果句柄
typedef struct ZYXTxn_T ZYXTxn_T;        // 事务句柄
typedef struct ZYXParams_T ZYXParams_T;  // 参数集句柄
```

所有句柄均为不透明指针。用对应的 `zyx_*` 函数创建，用对应的 `zyx_*_close` 函数释放。

## ZYXValueType

```c
typedef enum {
    ZYX_NULL   = 0,
    ZYX_BOOL   = 1,
    ZYX_INT    = 2,
    ZYX_DOUBLE = 3,
    ZYX_STRING = 4,
    ZYX_NODE   = 5,
    ZYX_EDGE   = 6,
    ZYX_LIST   = 7,
    ZYX_MAP    = 8
} ZYXValueType;
```

由 `zyx_result_get_type()` 和 `zyx_result_list_get_type()` 返回。用于判断应调用哪个读取函数。

| 值 | 说明 | 读取函数 |
|----|------|---------|
| `ZYX_NULL` | 空值 | 无需读取函数 |
| `ZYX_BOOL` | 布尔 | `zyx_result_get_bool()` |
| `ZYX_INT` | 整数 | `zyx_result_get_int()` |
| `ZYX_DOUBLE` | 浮点 | `zyx_result_get_double()` |
| `ZYX_STRING` | 字符串 | `zyx_result_get_string()` |
| `ZYX_NODE` | 图节点 | `zyx_result_get_node()` |
| `ZYX_EDGE` | 图边 | `zyx_result_get_edge()` |
| `ZYX_LIST` | 列表 | `zyx_result_list_*()` 系列函数 |
| `ZYX_MAP` | Map | `zyx_result_get_map_json()` |

## ZYXNode

```c
typedef struct {
    int64_t id;
    const char* label;   // 主标签，null 结尾
} ZYXNode;
```

由 `zyx_result_get_node()` 填充。`label` 指针由结果句柄持有，在 `zyx_result_close()` 前有效。

如需完整属性，使用 `zyx_result_get_props_json()`。

## ZYXEdge

```c
typedef struct {
    int64_t id;
    int64_t source_id;
    int64_t target_id;
    const char* type;    // 关系类型，null 结尾
} ZYXEdge;
```

由 `zyx_result_get_edge()` 填充。`type` 指针由结果句柄持有，在 `zyx_result_close()` 前有效。

## 字符串生命周期规则

| 函数返回 `const char*` | 有效期至 |
|------------------------|---------|
| `zyx_result_get_string()` | `zyx_result_close()` |
| `zyx_result_get_props_json()` | `zyx_result_close()` |
| `zyx_result_get_map_json()` | `zyx_result_close()` |
| `zyx_result_get_error()` | `zyx_result_close()` |
| `zyx_result_column_name()` | `zyx_result_close()` |
| `zyx_result_list_get_string()` | `zyx_result_close()` |
| `zyx_get_last_error()` | 同线程上的下一次 ZYX API 调用 |

**重要**：如需超出上述有效期保留字符串，请立即复制。
