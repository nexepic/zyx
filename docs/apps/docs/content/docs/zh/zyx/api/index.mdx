# API 参考

本节以当前源码为准，描述 ZYX 的公开嵌入式接口：

- C++ API：`include/zyx/zyx.hpp`
- C API：`include/zyx/zyx_c_api.h`

## 如何选择

- 使用 C++20 开发，优先选 C++ API（类型更完整、接口更自然）。
- 做 Rust/Go/Node 等 FFI 绑定，优先选 C API（句柄式接口、边界清晰）。

## 设计约定

- 查询执行统一返回 `Result`/`ZYXResult_T`，由调用方显式迭代结果集。
- 事务支持两种方式：
  - API 显式事务（`beginTransaction` / `zyx_begin_transaction`）
  - Cypher 事务语句（`BEGIN` / `COMMIT` / `ROLLBACK`）
- 所有文档以当前实现能力为准，不再使用过时的模板异常体系命名。
