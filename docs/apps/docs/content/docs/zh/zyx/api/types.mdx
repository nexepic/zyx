# 类型系统

头文件：`include/zyx/value.hpp`

## zyx::Value

`zyx::Value` 是 `std::variant`，用于查询参数和结果访问：

| 变体 | C++ 类型 | 说明 |
|------|----------|------|
| Null | `std::monostate` | 空值 |
| Boolean | `bool` | `true` 或 `false` |
| Integer | `int64_t` | 64 位有符号整数 |
| Float | `double` | 双精度浮点数 |
| String | `std::string` | UTF-8 字符串 |
| Node | `std::shared_ptr<Node>` | 图节点引用 |
| Edge | `std::shared_ptr<Edge>` | 图边引用 |
| 浮点向量 | `std::vector<float>` | 紧凑数值向量（常用于 embedding） |
| 字符串列表 | `std::vector<std::string>` | 字符串列表 |
| 列表 | `std::shared_ptr<ValueList>` | 异构值列表 |
| Map | `std::shared_ptr<ValueMap>` | 字符串键映射 |

### 用法

```cpp
// 构造查询参数
std::unordered_map<std::string, zyx::Value> params;
params["name"] = std::string("Alice");
params["age"] = int64_t(30);
params["active"] = true;
params["score"] = 95.5;

// 从结果中读取值
zyx::Value val = result.get("n.name");
if (std::holds_alternative<std::string>(val)) {
    std::string name = std::get<std::string>(val);
}
```

## zyx::Node

```cpp
struct Node {
    int64_t id;
    std::string label;                           // 首标签（向后兼容）
    std::vector<std::string> labels;             // 全量标签
    std::unordered_map<std::string, Value> properties;
};
```

| 字段 | 说明 |
|------|------|
| `id` | 内部节点 ID（由 `IDAllocator` 分配） |
| `label` | 主标签（列表中第一个） |
| `labels` | 该节点的所有标签 |
| `properties` | 键值属性映射 |

## zyx::Edge

```cpp
struct Edge {
    int64_t id;
    int64_t sourceId;
    int64_t targetId;
    std::string type;                            // 关系类型
    std::unordered_map<std::string, Value> properties;
};
```

| 字段 | 说明 |
|------|------|
| `id` | 内部边 ID |
| `sourceId` | 源节点 ID |
| `targetId` | 目标节点 ID |
| `type` | 关系类型（如 `"KNOWS"`） |
| `properties` | 键值属性映射 |

注意：Edge 使用 `type`（非 `label`）。Node 使用 `labels`（复数，可有多个）。

## graph::PropertyValue

查询引擎内部用于属性存储和 `QueryBuilder`：

```cpp
struct PropertyValue {
    using MapType = std::unordered_map<std::string, PropertyValue>;

    PropertyValue();                    // Null
    PropertyValue(bool value);
    PropertyValue(int64_t value);
    PropertyValue(double value);
    PropertyValue(const std::string& value);
    PropertyValue(std::vector<PropertyValue> vec);  // List
    PropertyValue(MapType map);                      // Map

    PropertyType getType() const;
    std::string toString() const;
};
```

| `PropertyType` | 说明 |
|----------------|------|
| `NULL_TYPE` | 空值 |
| `BOOLEAN` | `bool` |
| `INTEGER` | `int64_t` |
| `DOUBLE` | `double` |
| `STRING` | `std::string` |
| `LIST` | `std::vector<PropertyValue>` |
| `MAP` | `unordered_map<string, PropertyValue>` |
| `DATE` | `TemporalDate` — 日历日期（距 epoch 天数） |
| `DATETIME` | `TemporalDateTime` — 时间戳（距 epoch 毫秒数） |
| `DURATION` | `TemporalDuration` — ISO 8601 时长（months + days + nanos） |

## graph::ResultValue

```cpp
class ResultValue {

    std::string toString() const;

```

## 时间类型

头文件：`include/graph/core/TemporalTypes.hpp`

### TemporalDate

以距 1970-01-01 的天数存储的日历日期。

```cpp
struct TemporalDate {
    int32_t epochDays = 0;

    static TemporalDate today();
    static TemporalDate fromISO(const std::string& s);   // "YYYY-MM-DD"
    static TemporalDate fromYMD(int year, int month, int day);

    int year() const;
    int month() const;
    int day() const;
    std::string toISO() const;  // "YYYY-MM-DD"
};
```

### TemporalDateTime

以距 epoch 的毫秒数存储的时间戳。

```cpp
struct TemporalDateTime {
    int64_t epochMillis = 0;

    static TemporalDateTime now();
    static TemporalDateTime fromISO(const std::string& s);  // "YYYY-MM-DDTHH:MM:SS"
    static TemporalDateTime fromComponents(int y, int mo, int d, int h, int mi, int s, int ms = 0);

    int year() const;
    int month() const;
    int day() const;
    int hour() const;
    int minute() const;
    int second() const;
    std::string toISO() const;  // "YYYY-MM-DDTHH:MM:SSZ"
};
```

### TemporalDuration

ISO 8601 时长，包含三个独立分量：月、日、纳秒。

```cpp
struct TemporalDuration {
    int64_t months = 0;   // years * 12 + months
    int64_t days   = 0;   // weeks * 7 + days
    int64_t nanos  = 0;   // 时/分/秒折算为纳秒

    static TemporalDuration fromISO(const std::string& s);  // "P1Y2M3DT4H5M6S"
    static TemporalDuration fromComponents(int64_t years, int64_t months, int64_t weeks,
                                           int64_t days, int64_t hours, int64_t minutes,
                                           int64_t seconds, int64_t nanoseconds = 0);
    std::string toISO() const;
};
```

:::info
时间类型支持 Cypher 算术运算：`date + duration`、`datetime - duration`、`date - date`（返回 duration）等。可通过属性语法访问分量：`d.year`、`dt.hour`、`dur.months`。
:::

## 枚举类型

| 枚举 | 值 | 说明 |
|------|-----|------|
| `EntityType` | `Node`, `Edge`, `Property`, `Blob`, `Index`, `State` | 内部实体类型 |
| `OpenMode` | `OPEN_CREATE_NEW_FILE`, `OPEN_EXISTING_FILE`, `OPEN_CREATE_OR_OPEN_FILE` | 数据库打开模式 |
| `TxnState` | `TXN_ACTIVE`, `TXN_COMMITTED`, `TXN_ROLLED_BACK` | 事务状态 |
| `Language` | `Cypher` | 查询语言（当前仅 Cypher） |
