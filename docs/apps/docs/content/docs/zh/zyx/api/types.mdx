# 类型定义

头文件：`include/zyx/value.hpp`

## Value

`zyx::Value` 是 `std::variant`，当前包含：

- `std::monostate`（NULL）
- `bool`
- `int64_t`
- `double`
- `std::string`
- `std::shared_ptr<Node>`
- `std::shared_ptr<Edge>`
- `std::vector<float>`（常用于 embedding）
- `std::vector<std::string>`

## Node

`zyx::Node` 字段：

- `id`
- `label`（首标签，兼容字段）
- `labels`（全量标签）
- `properties`

## Edge

`zyx::Edge` 字段：

- `id`
- `sourceId`
- `targetId`
- `label`
- `properties`

## 类型建议

- 向量检索属性建议优先使用 `std::vector<float>`。
- 通用列表若需要数值精度控制，建议在业务层统一序列化策略。
