# 缓存管理

当前缓存不是单一模块，而是多处协同：

- 通用页缓存：`PageBufferPool`
- 若干组件内部 LRU（如向量索引 registry）
- DirtyEntityRegistry（写路径脏集）

## 设计目标

- 降低重复磁盘读
- 在事务边界前收敛写放大
- 让批量写在内存受控范围内进行

## Dirty 与 flush

`PersistenceManager` 维护多类实体的 dirty registry，并在达到阈值时触发 auto-flush 回调。

关键点：

- 阈值按“所有实体类型总量”计算
- flush 后提交快照，避免部分类型积压导致内存失控
