# 基准测试

ZYX 提供独立 benchmark 应用（`apps/benchmark`），覆盖插入、查询、并发、向量检索与优化器场景。

## 覆盖场景

- Insert：Cypher 与 Native 路径、不同索引配置
- Query：属性索引、标签扫描、全扫描
- Algo：最短路径
- Vector：向量写入与 ANN 查询
- Concurrency：多线程读写/排序/向量检索
- Optimizer：规则收益对比

## 代码位置

- 入口：`apps/benchmark/main.cpp`
- 框架与配置：`apps/benchmark/BenchmarkFramework.hpp`、`BenchmarkConfig.hpp`
- 各场景：`BenchmarkStorage.hpp`、`BenchmarkVector.hpp`、`BenchmarkOptimizer.hpp` 等

## 结果解读建议

- 先固定硬件、线程数、数据规模
- 分开看冷启动与稳态数据
- 结合 profile/explain 与索引命中率一起分析
- 对比版本时保持同一编译配置与数据分布
