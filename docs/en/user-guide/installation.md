# Installation

## Prerequisites

- C++20 compatible compiler (GCC 11+, Clang 13+, MSVC 2022+)
- Meson build system
- Conan package manager
- Ninja build backend

## Build from Source

```bash
# Clone the repository
git clone https://github.com/yuhong/metrix.git
cd metrix

# Install dependencies and build
./scripts/run_tests.sh
```
