# Rename Metrix to ZYX Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Rename the entire project from "metrix" to "zyx" (display name "ZYX") across build system, public API, C API, documentation, and user-facing strings.

**Architecture:** Use Meson's `configure_file()` to generate a centralized configuration header (`ProjectConfig.hpp`) from a template file. This provides a single source of truth for project name constants used throughout the codebase.

**Tech Stack:** Meson build system, C++20, Conan package manager, Google Test framework
